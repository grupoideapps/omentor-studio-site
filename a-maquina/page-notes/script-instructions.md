# Script Instructions

## Product lockup rules

1. Keep the lockup as a reusable component.
2. Preserve the transparent background.
3. Keep the outer aspect ratio stable.
4. Change the icon only through the icon SVG inside the badge.
5. Build the icon as a semantic brand glyph, inspired by Iconify-style minimal SVGs, not as a decorative star by default.
6. Keep the icon visually aligned with the bottom text line.
7. Adjust the vertical spacing between lines before changing overall scale.
8. If the icon becomes too small, enlarge the inner SVG first, then the badge container.

## Editing notes

- Do not switch the page away from the design system font stack.
- Prefer inline SVG for the icon shape when the icon needs to be original.
- Keep future icon changes semantically tied to the product name or category.
- Use the hero version and price version of the lockup as the same source of truth.
